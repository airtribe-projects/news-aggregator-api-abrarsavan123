Brief
In this guided project, you will build a RESTful API for a personalized news aggregator using Node.js, Express.js, bcrypt, and JWT. You'll implement authentication, user preferences, and external API integration while practicing error handling, input validation, and token-based security.

By completing each step thoroughly, you'll develop a solid understanding of API development in Node.js and gain practical experience with real-world scenarios like caching, user authentication, and news article management.

Focus on understanding the concepts step-by-step and implementing them in your own way.

Good luck!



Some external news APIs that may be helpful (they have some limits on their free tier plans):

NewsAPI - 100 requests/day (this one has a native Node.js library that is unmaintained since 2018. So it's better to consume the APIs directly)

NewsCatcher News API

GNews API - 100 requests/day

NewsAPI.ai - 2000 requests/month

Note: All of them require you to sign up for the API key.

Submission guidelines
Accept the Github classroom assignment here: https://classroom.github.com/a/qq74h_eC

Develop your APIs according to the steps mentioned in the brief.

Write a clear and concise README file.

Run npm run test and confirm all test cases are passing.

To submit your project, please provide the PR link to the feedback branch (This will be automatically created in your repo once you accept the assignment. Ensure that you push your code to the main branch and submit PR #1 as the submission).
